*********************************************
Data Writers (:mod:`pyvisgen.io.datawriters`)
*********************************************

.. currentmodule:: pyvisgen.io.datawriters

Data writer submodule of :mod:`pyvisgen.io`. Data writers allow writing
data simulated with ``pyvisgen`` to different formats. All data writers
are subclasses of the abstract base clase :class:`~pyvisgen.io.datawriters.DataWriter`
and support context manager protocol.
